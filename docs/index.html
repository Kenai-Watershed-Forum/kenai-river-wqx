Preface

This document is the online work-in-progress version of the Water Quality Assessment of the Kenai River Watershed, 2000 - 2023. When completed, it will be an update to historical reports summarising available data to date for this project (McCard 2007; Guerron Orejuela 2016).

To see a ten-minute presentation introducing this project and it’s history, watch the following YouTube video below:


This report can also be downloaded as an Microsoft Word document by clicking on the ‘W’ symbol in the upper left corner. When the report is complete, it will be available as a PDF. All project files and source code are hosted in a public GitHub repository, which is accessible by clicking the GitHub symbol in the upper left corner.

This online report is not optimized for use on mobile devices, and is best accessed on a desktop or laptop computer.


Guerron Orejuela, Edgar. 2016. “Water Quality Assessment of the Kenai River Watershed from July 2000 to July 2014.” https://dec.alaska.gov/media/16756/kenai-river-baseline-monitoring-report-final-zncuappendix.pdf.
McCard, Jennifer J. 2007. “Water Quality Assessment of the Kenai River Watershed from July 2000 to July 2006.”
